Authenticator:
TheAltening's authenticator is used to gain access to our servers and our alt tokens, it is necessary to use the alts and can be installed in less than a minute.
How to set up the authenticator:
1. Download the application by clicking the "Download" button below.
2. Extract the downloaded files. [example]
3. Launch "authenticator.exe" as administrator. [example]
4. Enter your API Key. [example] [api-key]
How to use alt tokens:
1. Switch authentication servers. [example]
2. Launch minecraft.
3. Enter the alt token in 'Email or username' field and any password. [example]
